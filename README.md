# 一个干净的React实例

### 我们萌新学React的时候往往都会发现
### React的示例工程有好多示例文件，所
### 以我觉得给大家搞了一个干净的React
### 实例，没有什么乱七八糟的东西，就一
### 个主类和App实例类，我还添加了一些
### 中文注释，供大家看懂。

### 输入以下指令启动
``` bash
npm start
```

### 输入以下指令补全依赖
``` bash
npm install
```

### 输入以下指令来编译构建
``` bash
npm run build
```

### 懒人的话可以以下操作

### Linux & MacOSX
``` bash
bash build.sh
```

### Windows
``` bash
build.bat
```
